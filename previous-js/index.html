What's in Previous.js? All the tools you need to make the Web. Faster.

Rendering
Render contents using pages and nested layouts within your app routing structure and load resources dynamically only when necessary.
Routing
Create routes using the file system. Navigate with usual HTML links and instantly stream UI from the server with jQuery Ajax.
jQuery DOM Manipulation
Manipulate DOM Nodes without sending additional client-side JavaScript. Built on the latest jQuery features.
Data
Make your application synchronously await data fetching for page rendering in both server and client side.
Components
Create components to render contents both server and client side using the same code.
SEO Friendly
Setup your page headers, including title and metadata tags, to provide SEO friendly information in your application pages.
APIs
Make API endpoints to securely connect with third party services and consume from your frontend.
Publishing
